# **App Name**: CyberVertex PUTTI POOJITHA NANDA SAI BHAGYA GANESH

## Core Features:

- Cinematic Loading Sequence: A high-impact, performance-optimized loading screen featuring procedural geometry and cinematic animations to captivate users immediately.
- Astral Hero Environment: An immersive Three.js scene with a 3D Earth, rotating particles, animated shooting stars, and an interactive floating laptop model that reacts to mouse movement.
- Kinetic Camera System: Smooth, scroll-triggered camera movements that navigate a 3D space, creating a seamless narrative flow between portfolio sections.
- Holographic Project Deck: Awwwards-quality project showcase using 3D glassmorphic cards with interactive tilt effects and direct links to GitHub and live demos.
- Developer Analytics Hub: A specialized dashboard integrating a live GitHub contribution graph alongside LeetCode and HackerRank statistics for real-time technical validation.
- Intelligent Recruiter Assistant: An AI-powered tool designed to answer complex natural language questions about Ganesh's development experience and skill proficiency.
- Quantum Resume Portal: A sleek, high-conversion contact section featuring a neon-styled resume download button and smooth entrance transitions.

## Style Guidelines:

- Ultra-modern dark theme using a pitch-black background (#000000) illuminated by neon blue, electric cyan, and deep purple glow effects.
- Dynamic FPS-optimized transitions including mouse parallax, particle systems, and spring-physics camera movements for a premium feel.
- Glassmorphism UI design with high-end visual hierarchy, generous negative space, and a mobile-responsive spatial architecture.
- Header font: 'Space Grotesk' for a high-tech vibe; Body font: 'Inter' for maximum readability; Monospace: 'Source Code Pro' for technical data.
- Minimalist linear SVG icons with soft outer glow filters in cyan and purple to maintain the cyberpunk aesthetic.
